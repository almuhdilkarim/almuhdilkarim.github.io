Mengenal Slock - Screen Saver Linux Yang Sederhana Namun Full Performance

Slock adalah salah satu aplikasi yang berguna untuk menampilkan screensaver atau mengunci antar muka (Screen locker ) dari suatu sistem linux supaya tidak bisa digunakan oleh orang lain selama perangkat ditinggalkan dalam keadaan hidup.

Slock sendiri adalah kependekan dari “Simple X display locker”. Dalam perancanganya, slock didesain dengan secara minimalis. Karena didesain secara minimalis, maka jangan berharap anda akan mendapatkan tampilan yang menawan seperti aplikasi screen saver lainya.

Ketika layar terkunci oleh slock, secara default slock hanya akan membuat tampilan layar menajadi hitam (blank). Ketika anda mengetikan sesuatu pada keyboard, warna hitam tersebut akan berubah menjadi warna merah dan biru.

Walupun demikian, slock bisa bekerja dengan sangat cepat dan ramah dalam penggunaan memory. Disamping itu, karena slock didesain secara minimalis, maka jika ada celah keamanan dari aplikasi cendrung akan lebih akan lebih mudah dan cepat untuk diperbaiki.

Instalasi Slock di Arch Linux

  • Buka terminal emulator
  • Update repository menggunakan perintah
	sudo pacman -Syy
  • Setelah itu baru Ketikan perintah berikut untuk memulai installasi :
	sudo pacman -S slock

Tunggu beberapa saat, sampai proses installasi selesai.

Instalasi Slock di Linux Ubuntu

  • Buka terminal emulator
  • Update repository menggunakan perintah
	sudo apt-get update -y
  • Setelah itu baru Ketikan perintah berikut untuk memulai installasi :
	sudo apt-get install -y slock

Tunggu beberapa saat, sampai proses installasi selesai.

Instalasi Slock di Linux Fedora

  • Buka terminal emulator
  • Update repository menggunakan perintah
	sudo yum -y update
  • Setelah itu baru Ketikan perintah berikut untuk memulai installasi :
	sudo yum -y install slock

Tunggu beberapa saat, sampai proses installasi selesai.

Cara Mengunci Layar Menggunakan Slock

  • Buka terminal emulator
  • Ketikan perintah berikut :
 slock

Ketika layar anda akan menjadi hitam (Blank), Berarti layar anda telah terkunci oleh slock

Cara Membuka Layar Yang Terkunci

Untuk membuka layar yang terkunci oleh slock, anda cukup,

  • Menekan tombol apa saja
  • Ketika warna layar berubah, ketikan password anda.

Mungkin ini adalah salah satu faktor yang mebuat slock menjadi “simplest screen locker” . Ketika aplikasi screen locker lain membutuhkan username dan password untuk membuka layar, pada slock, kita cukup menginput password untuk membuka kunci layar.

Simplisitas yang ditawarkan oleh slock ini yang membuat saya menjadikan slock sebagai screen locker favorit saya ( karena saya cukup merasa malas mengetikan username, hehe).

Tips : Otomasi Penguncian Layar Menggunakan Systemd

Dalam langkah sebelumnya, untuk mengunci layar,kita perlu membuka terminal terlebih dahulu. Walaupun tidak terlalu signifikan, namun bagi saya ini adalah hal yang merepotkan,

Nah, dalam tips ini, kita akan membuat slock secara otomatis mengunci layar, ketika keadaan layar dalam keadaan sleep. Sehingga kita tidak perlu mengulang langkah, dengan membuka terminal dan mengetik perintah slock ketika akan megunci layar. Pekerjaan yang membosankan bukan? hehe.

Untuk otomasi slock, kita akan membuat sebuah configurasi sederhana pada systemd. berikut langkahnya,

  • buka terminal
  • ketikan perintah
sudo su
  • Buat file konfigurasi slock pada systemd menggunakan perintah
nano /etc/systemd/system/slock@.service
  • Copy configurasi berikut
[Unit]
Description=Lock X session using slock for user %i
Before=sleep.target

[Service]
User=%i
Environment=DISPLAY=:0
ExecStartPre=/usr/bin/xset dpms force suspend
ExecStart=/usr/bin/slock

[Install]
WantedBy=sleep.target
  • Selanjutnya save configurasi slock@.service dengan menekan tombol Ctrl+X, ketika muncul dialog ketikan Y
  • Restart perangkat anda atau jika anda malas untuk merestart perangkat, anda bisa mengetikan perintah berikut pada terminal anda.
systemd daemon-reload

Setelah itu, slock secara otomatis akan mengunci perangkat ada ketika posisi perangkat anda idle dan screen anda memasuki posisi sleep.
