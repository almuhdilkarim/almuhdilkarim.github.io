Sejak perbincangan tersebut, tidak ada lagi perbincangan lanjutan terkait bagaimana khayalan tersebut menjadi nyata. 
Pada tahun 2010, semester kedua perkuliahan saya, saya dibelikan suatu unit komputer rakitan oleh kedua orang tua saya.

Bagi saya, ini adalah salah satu momen yang paling membahagiakan. Saya bisa mulai membuat tugas di kosan, 
Karena sebelumnya saya mengetik tugas di rental ataupun warnet. 

Namun, semiggu setelah komputer baru tersebut hadir di kamar kosan saya. Komputer yang menggunakan sistem operasi
windows XP tersebut mengadakan acara halal bi halal dengan mallware ( baca - virus komputer ). Mallwarenya biasa, cuma file .lnk. Walaupun cuma
mallware biasa, hal tersebut membuat saya gundah, maklum komputer baru..

Kekesalan saya, membuat saya menaiki angkot menuju Poins Square. Tujuan saya adalah mencari CD/DVD sistem operasi yang tahan terhadap
virus. 

Di toko tersebut, saya melihat DVD RHEL ( Red Hat Enterprise LInux). Saya pernah mendengar dari salah seorang teman papa saya,
seorang pengembang program, kalau linux itu lebih aman dari virus. 

Tanpa fikir panjang, saya langsung membeli DVD tersebut dan pulang ke kosan untuk mengganti sistem operasi saya dengan dengan RHEL.
Setelah memasukan DVD kedalam DVD room, saya mulai boot kedalam proses installasi.

Saya fikir masalah sudah selesai, akan tetapi ternyata tampilan installasi RHEL berbeda dengan dengan Windows XP. 
Saya kebingungan bagaimana cara menginstall sistem operasi Red Hat. 


Saya kemudian beranjak kembali ke warnet. di warnet saya search cara installasinya, kemudian ngeprint setiap tutorial installasi yang ada
di internet. Tentu banyak yang bertanya, kenapa tidak pakai disket (Saat itu harga flashdisk masih mahal, malah flashdisk di jadikan mainan kalung dan terlihat keren)? 
Saya tidak mau menggunakan disket,karena takut virus akan tertinggal disket.. hehe.. fikiran bodoh yang lucu untuk saya hari ini.

setelah 4 kali,bolak balik warnet untuk mencari dan ngeprint tutorial. Akhirnya komputer saya berhasil di install RedHat.. Samapai di sini saya lega,
saya merasa komputer saya akan aman. Selanjutnya saya ingin menginstall Micorosoft Office 2007, winamp dan aplikasi aplikasi  lainya.

Setelah memasukan cd installasi kedalam dvd room, dan mengklik dua kali icon instalasi. Tiba tiba pesan error muncul. 
Saya coba ganti cd office dengan meminjam ke warnet, awalnya saya berfikir cd saya telah rusak (baca - corrupt).

Installasi dari cd warnet juga menampilkan pesan error yang sama.Saya masih berfikir cd warnet rusak.
kemudian saya mengganti dengan cd installasi aplikasi winamp. PEsan error yang sama masih saya temukan.

Kesimpulan saya selanjutnya adalah, cd installan redhat yang rusak. Lebih parahnya, tidak ada ketentuan pengembalian di toko tersebut.
tidak mungkin saya mengeluarkan 50.000 lagi untuk beli DVD baru, karena bagi mahasiswa perantauan 50.000 adalah jumlah uang yang cukup
besar.

di akhir hari tersebut, saya kembali menginstall komputer saya dengan windows XP beserta kelengkapan aplikasi lainya. Ajaibnya, seluruh 
cd installan saya bekerja dengan baik, tidak menampikan error sama sekali. Dari situ saya mengambil kesimpulan dengan keyakinan penuh, 
DVD Installan RHEL saya yang rusak..

Kesini-sininya, ketika mengingat kembali momen momen tersebut, saya seringkali tertawa kecil.. Karena saya yakin, saya tidak 
sendirian, pasti ada linuxer lain yang memiliki pengalaman yang hampir sama, apalagi notabenenya jalur yang ditempuh adalah
jalur otodidak.
