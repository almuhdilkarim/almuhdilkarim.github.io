Mengetahui Single Version PHP

Buka terminal pada linux ubuntu anda, kemudian ketikan perintah,

php --version

Setelah itu akan keluar output sebagai berikut.

PHP 8.0.1 (cli) (built: Jan 13 2021 08:22:35) ( NTS )
Copyright (c) The PHP Group
Zend Engine v4.0.1, Copyright (c) Zend Technologies
    with Zend OPcache v8.0.1, Copyright (c), by Zend Technologies

Dari keterangan tersebut, kita mengentahui kalau versi php yang sedangkan digunakan adalah versi PHP 8.0.1

Mengatahui Multiple Version PHP

Dalam beberapa kasus, terkadang kita menginstall berbagai versi php untuk beberapa kebutuhan khusus.

Untuk mengetahui semua versi php telah terinstall di dalam server atau komputer local, kita bisa menggunakan perintah locate.

Pertama buka terminal anda, atau login ke server anda dengan menggunakan ssh. Selanjutnya, silahkan ketikan perintah berikut ini di terminal dan enter,

locate bin/php

Sebagai contoh, dikomputer saya keluar output sebagai berikut,

/usr/bin/php
/usr/bin/php8.0
/usr/local/lsws/lsphp73/bin/php
/usr/local/lsws/lsphp73/bin/php-helper
/usr/local/lsws/lsphp73/bin/php-maintscript-helper
/usr/local/lsws/lsphp73/bin/php7.3
/usr/local/lsws/lsphp73/bin/phpdismod
/usr/local/lsws/lsphp73/bin/phpenmod
/usr/local/lsws/lsphp73/bin/phpquery
/usr/sbin/php-fpm8.0
/usr/sbin/phpdismod
/usr/sbin/phpenmod
/usr/sbin/phpquery

Output dari perintah tersebut akan menampilkan versi php yang terinstall didalam perangkat anda.

Hasil diatas adalah output dari perangkatsaya. pada baris kedua kita menemukan php8.0 dan pada baris ke-3 sampai ke-9 kita menemukan lsphp73. terakhir pada barus ke-10 kita menemukan php-fpm8.0.

Berdasarkan contoh diatas, dapat diketahui kalau saya menggunakan php8.0, lsphp7.3 dan php-fpm8.0.
