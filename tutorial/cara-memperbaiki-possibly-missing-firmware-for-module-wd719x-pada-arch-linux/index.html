Memperbaiki Possibly Missing Firmware for Module: Wd719x Pada Arch Linux

Published
Apr 24, 2021
Last Update
Jun 9, 2021
Reading
Estimate 1-min

  • Buka terminal

  • Masuk kedalam folder /tmp, dengan mengetikan perintah berikut di terminal

    cd /tmp

  • Clone file firmware dari github

    git clone https://aur.archlinux.org/wd719x-firmware.git

  • Masuk kedalam folder firmware

    cd wd719x-firmware

  • Install firmware

    makepkg -sri

  • Update konfigurasi mkinitcpio

    sudo mklinitcpio -p linux

  • Restart

    systemd reboot
