Memperbaiki Possibly Missing Firmware for Module: Aic94xx Pada Arch Linux

Al Muhdil Karim
Published
Jun 12, 2021
Last Update
Jun 26, 2021
Estimate
1-min read

Buka terminal

Masuk kedalam folder /tmp, dengan mengetikan perintah berikut di terminal

cd /tmp

Clone file firmware dari github

git clone https://aur.archlinux.org/aic94xx-firmware.git

Masuk kedalam folder firmware

cd aic94xx-firmware

Install firmware

makepkg -sri

Update konfigurasi mkinitcpio

sudo mkinitcpio -p linux

Restart perangkat dengan perintah

systemd reboot
